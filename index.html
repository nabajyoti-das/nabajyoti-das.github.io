The Closed Window

Written on July 5, 2017

It was beautiful, pleasant, dark and deep,
when the window was open.

There was happiness, togetherness, calmness and peace,
when the window was open.

But now the window is closed,
So everything is lost.

I try to peep through the window
But all I see is a vague picture
Which fades away just like a mist,
when the sun rise up.

But now the window is closed
No matter how much I try
I still couldn't see through the window again.

I sometimes think what lies behind
the closed window.
But all I see is emptiness.
